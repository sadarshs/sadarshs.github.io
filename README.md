# sadarshs.github.io

This portfolio is set up to deploy to GitHub Pages from the `main` branch using GitHub Actions.

## Local development

```bash
npm install
npm run dev
```

## Build locally

```bash
npm run build
```

## Publish to GitHub Pages

1. Push this project to the `main` branch of the `sadarshs/sadarshs.github.io` repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Set `Source` to `GitHub Actions`.
4. After each push to `main`, GitHub will build and deploy the site automatically.

Your site URL will be:

```text
https://sadarshs.github.io/
```
